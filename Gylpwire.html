Gylpwire
Interaction Designer Visual Designer Project Manager
Overview

Glyphwire is a collection of monochromatic glyphs or icons for designers to use in their projects. It started as an internal project, as a repository to store all icons created by the design team. Worked as a Interaction, Visual Designer and project manager in developing this project. Coordinated with other designer and developer in getting the work done.Understood the basics in responsive web design and development. Learnt a lot about people management.

To see the site http://www.glyphwire.com/

The Problem

Every designer in the company had their own set of Icons and not interested in sharing it with other designer. The icon set created for a specific project rests only with the person who designed it and other people cannot use it if their project needs a similar one. And developers have to ask designers for icons every time when they want one.

The Challenge

The challenge that I found in this project is actually more about handling people than solving problem. The problems challenge is simple. Make individual designer to contribute to the benefit of the whole company. But scaling it up was a real challenge.

The Research

Not much research was done to create the application that caters people only within company. But when we wanted to scale up I did a competitor analysis. I looked into various websites that provide icons like thenounproject etc. I found that this is a saturated market and the only viable model was icon on demand. But the company was not interested in pursuing it and so it remain as a curated repository of Icons for sharing within the company and also as a display for the client.

Design Stories
Analysis

My idea was to make it function as simple as possible. So I wanted it to be a curated website where there will be some control over the standard of Icon being uploaded into the site. As for as download is concerned it's made much simpler with variety of sizes available in PNG along with an SVG to do any kind of editing with it. Anyone can see the icons but only an person within the company can download using his login credentials. By this way we made sure that other people can visually see the icons but cannot download and use it.
